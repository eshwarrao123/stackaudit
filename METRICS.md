# Product Metrics & Instrumentation

## North Star Metric
**Audit Completion Rate (ACR)**: The percentage of users who start the audit form and reach the final "Report" page.
- **Why it matters**: This metric confirms that our UI is frictionless and the "Value-to-Effort" ratio is high enough to keep busy founders engaged.

## Input Metrics
1. **Lead Conversion Rate (LCR)**: Percentage of report viewers who provide an email for "Optimization Alerts." This measures the perceived long-term value of the product.
2. **Viral Coefficient (k)**: The average number of new users generated by a shared "Audit Report" link. This measures our primary distribution channel's effectiveness.
3. **Average Identified Waste (AIW)**: The dollar amount of waste found per audit. If this falls below $30, the "pain point" is too small, and we need to target larger teams.

## Instrumentation Priorities
- **Funnel Drop-off**: Track precisely which step of the 4-step audit form users abandon (Tools vs. Spend vs. Usage).
- **CTA Click-Throughs**: Monitor the "Copy Share Link" and "Consolidation Guide" clicks to see which report sections are most actionable.
- **AI Fallback Frequency**: Log how often the system falls back to deterministic summaries due to OpenAI API errors to ensure infrastructure stability.

## Pivot Trigger Metric
**The "So What?" Metric**: If less than 5% of users with an Optimization Score below 70 click the "How to Consolidate" button, it indicates that "knowing" about waste isn't enough of a motivator. 
- **Pivot**: Move from a "Diagnostic Tool" to an "Automated Cleanup Tool" that handles the cancellations via a Chrome Extension or direct API integrations.
