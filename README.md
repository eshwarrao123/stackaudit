# StackAudit

StackAudit is an AI spend optimization platform that helps startups analyze and reduce unnecessary spending on AI tools like ChatGPT, Claude, Cursor, GitHub Copilot, Gemini, and OpenAI APIs.

The platform generates personalized audit reports with optimization recommendations, estimated savings, and actionable insights for engineering and product teams.

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- shadcn/ui
- Supabase
- React Hook Form
- Zod
- Framer Motion
- Recharts
